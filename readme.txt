This repository is related to finished, ongoing and upcomming projects. Each project is not owned by me, bit is a collaborative project. Member collaborating in each project are listed in each project folder readme file. For each collaborator some thing is promissed in return, and it is required to produce that thing in return.

Structure of readme of each project folder is :

--[ 1st : TO BE COMPLETED ]--

This section will outline final objectives that are expected to be accomplished.

--[ 2nd : TEAM MEMBERS ]--
this section will identify the team members.

--[ 3rd : REQUESTED TASKS ]--

this section will outline tasks in association with collaborators/member. structure is as follows

Task name: Member/Contributor : Task Description
